### Installation

The library contains the following modules

- [Core](core/README.md)
- Proto
- [Spring](spring/README.md)

To use Tabour only **core** is required. Proto and Spring packages contain helpers and functionality regarding

- Protobuf
- Spring Boot

## Supported messaging systems

- SQS
